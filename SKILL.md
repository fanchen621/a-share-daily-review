# A股全时段智能复盘系统 v6

## 架构概览

三个定时任务，每个交易日自动执行：

| 时间 | 任务 | 内容 |
|------|------|------|
| 08:00 | 盘前速递 | 外盘/大宗/新闻 → 判断今日操作方向 |
| 12:00 | 午盘策略 | 上午数据 → 选股建议 + 详细分析 |
| 17:30 | 收盘复盘 | 全量数据 → 复盘分析 + 经验更新 |

## 数据源架构

```
纯API数据源（不依赖 mimo_web_search）:
  ├─ 腾讯行情API (qt.gtimg.cn) → A股指数 + 美股指数 + 美股个股 + 商品期货 + 恒生期货
  ├─ 财联社API (cls.cn) → 电报/快讯/财经新闻
  ├─ 东方财富数据中心 (datacenter-web) → 龙虎榜 + 北向资金
  └─ 汇率API (exchangerate-api) → 主要货币汇率
```

**腾讯行情覆盖:**
- A股: 上证指数/深证成指/创业板指/科创50/沪深300/上证50
- 美股: 道琼斯/纳斯达克/标普500/纳斯达克100
- 美股个股: 英伟达/特斯拉/苹果/微软
- 商品: 纽约金/纽约银/WTI原油
- 亚太: 恒生指数期货

**东方财富数据中心覆盖:**
- 龙虎榜: 每日上榜个股+买卖席位+净买入额
- 北向资金: 近5日净流入数据

## 任务一：盘前速递 (08:00)

### 执行流程

```bash
cd ~/.openclaw/skills/a-share-daily-review
python3 scripts/a_share_scraper.py --mode premarket -o data/$(date +%Y%m%d)_premarket.md
```

脚本自动采集以下数据：
1. 美股三大指数 + 纳斯达克100
2. 美股权重股：英伟达/特斯拉/苹果/微软
3. 商品期货：纽约金/纽约银/WTI原油

## Cron 定时任务配置

### 盘前 (08:00)
```
任务名: a-share-premarket
cron: 0 8 * * 1-5
sessionTarget: isolated
```

**AgentTurn 指令:**
```
执行A股盘前速递:
1. 运行: python3 scripts/a_share_scraper.py --mode premarket -o data/$(date +%Y%m%d)_premarket.md
2. 读取报告，补充分析判断（每个数据组后附📊分析）
3. 读取 data/lessons.md 了解近期盘感
4. 输出完整盘前速递
```

### 午盘 (12:00)
```
任务名: a-share-midday
cron: 0 12 * * 1-5
sessionTarget: isolated
```

**AgentTurn 指令:**
```
执行A股午盘策略:
1. 运行: python3 scripts/a_share_scraper.py --mode midday -o data/$(date +%Y%m%d)_midday.md
2. 读取今天的盘前报告进行对比
3. 读取 data/lessons.md
4. 给出下午操作策略和选股建议
```

### 收盘 (17:30)
```
任务名: a-share-closing
cron: 30 17 * * 1-5
sessionTarget: isolated
```

**AgentTurn 指令:**
```
执行A股收盘复盘:
1. 运行: python3 scripts/a_share_scraper.py --mode closing -o data/$(date +%Y%m%d)_closing.md
2. 读取盘前和午盘报告进行对比复盘
3. 读取 data/lessons.md
4. 生成完整复盘分析（量价关系、板块轮动、龙虎榜分析、明日预判）
5. 更新 data/lessons.md（板块联动、情绪周期、量化动向、判断修正）
```

## 关键原则

1. **纯API驱动**: 所有数据通过稳定API获取，不依赖搜索引擎
2. **数据完整性**: 腾讯行情覆盖A股/美股/商品/期货，龙虎榜+北向通过东方财富
3. **盘感是灵魂**: lessons.md 是核心资产，每次收盘必须更新
4. **列表适配钉钉**: 所有数据用列表展示，不用表格
5. **每组数据必有分析**: 数据后面必须跟分析段落
6. **诚实面对失误**: 预判错了记下来，这是成长的唯一途径
7. **输出要可操作**: 不是罗列数据，而是「该买什么、卖什么、等什么」
