# A股全时段智能复盘系统 v5

## 架构概览

三个定时任务，每个交易日自动执行：

| 时间 | 任务 | 内容 |
|------|------|------|
| 08:00 | 盘前速递 | 外盘/大宗/比特币/新闻 → 判断今日操作方向 |
| 12:00 | 午盘策略 | 上午数据 → 选股建议 + 详细分析 |
| 17:30 | 收盘复盘 | 全量数据 → 复盘分析 + 经验更新 |

## 数据源架构

```
稳定数据源:
  ├─ 东方财富 ulist API → 大盘指数(上证/深证/创业板/科创50) ✅已验证
  ├─ 财联社 API → 电报/快讯/财经新闻 ✅已验证
  └─ mimo_web_search → 所有其他数据 ✅最可靠

mimo_web_search 覆盖范围（必须全部搜索）:
  ├─ 美股: 道琼斯/纳斯达克/标普500 + 英伟达/特斯拉/苹果/微软股价
  ├─ 亚太: 日经225/韩国KOSPI/恒生指数
  ├─ 大宗商品: 伦敦金/伦敦银/布伦特原油/碳酸锂
  ├─ 加密货币: 比特币/以太坊
  ├─ 社交言论: 马斯克最新推特/特朗普最新政策
  ├─ A股详情: 涨跌家数/涨停跌停/板块涨幅/资金流向/龙虎榜/北向资金
  └─ 期货: A50期货/IF主力合约

scraper 输出:
  脚本生成报告框架(含{{mimo_web_search: "搜索词"}}占位符)，
  AI读取后用 mimo_web_search 填充所有占位符，生成最终报告。
```

## 关键原则

1. **数据完整性 > 分析速度**: 每天必须有数据，多源降级兜底

3. **列表适配钉钉**: 所有数据用列表展示，不用表格
4. **每组数据必有分析**: 数据后面必须跟「> 📊 分析」段落
5. **诚实面对失误**: 预判错了记下来，这是成长的唯一途径
6. **输出要可操作**: 不是罗列数据，而是「该买什么、卖什么、等什么」
7. **mimo_web_search 是关键**: 外盘/大宗/比特币/社交言论必须通过搜索获取
