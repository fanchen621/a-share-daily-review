# A股全时段智能复盘系统 v7

> OpenClaw 自动化 A 股复盘分析，覆盖盘前→午盘→收盘，钉钉适配列表格式。

## 数据源

| 来源 | 数据 | 状态 |
|------|------|------|
| 腾讯行情API | A股指数、美股指数、美股7只权重股、纽约金/银/原油、恒生期货 | ✅ 稳定 |
| 新浪期货API | 布伦特原油、日经225、比特币 | ✅ 稳定 |
| 财联社API | 电报/快讯/新闻 | ✅ 稳定 |
| 东方财富数据中心 | 龙虎榜、北向资金 | ✅ 稳定 |
| 汇率API | USD/CNY/JPY/KRW | ✅ 稳定 |
| mimo_web_search | KOSPI、以太坊、A50期货、碳酸锂、马斯克特朗普 | 运行时补充 |

## 报告内容

**盘前速递 (08:00)**
- 美股三大指数 + 英伟达/特斯拉/苹果/微软/谷歌/亚马逊/Meta
- 日经225 + 恒生期货 + KOSPI
- 纽约金/银 + WTI + 布伦特 + 碳酸锂
- 比特币 + 以太坊
- 马斯克 & 特朗普最新言论
- 财联社热点 + A50期货 + 盘前判断

**午盘策略 (12:00)**
- 上午行情回顾 + 风格判断 + 量能分析
- 龙虎榜机构行为
- 下午操作建议 + 重点标的推荐

**收盘复盘 (17:30)**
- 全天行情 + 量价分析
- 龙虎榜量化席位分析
- 北向资金近5日趋势
- 明日预判 + 操作建议
- 盘感数据库更新

## 快速开始

```bash
# 盘前
python3 scripts/a_share_scraper.py --mode premarket

# 午盘
python3 scripts/a_share_scraper.py --mode midday

# 收盘
python3 scripts/a_share_scraper.py --mode closing
```

## Cron 定时

| 时间 | 任务 |
|------|------|
| 08:00 | `0 8 * * 1-5` 盘前速递 |
| 12:00 | `0 12 * * 1-5` 午盘策略 |
| 17:30 | `30 17 * * 1-5` 收盘复盘 |

scripts/a_share_scraper.py   # 数据采集 v7
